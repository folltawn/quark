# Quark (EN)
A compiled low-level programming language.

![platform Windows](https://img.shields.io/badge/Platform-Windows-blue)

It is **NOT**:
* A high-level language
* An interpreted language
* A virtual machine
* ​​A DSL (Domain-Specific Language)

It is:
* A low-level language
* A compiled language

# Quark (RU)
Компилируемый низкоуровневый язык программирования.

![platform Windows](https://img.shields.io/badge/Platform-Windows-blue)

**НЕ** является:
* Высокоуровневым языком
* Интерпретируемым языком
* Виртуальной машиной
* DSL (Domain-Specific Language)

Является:
* Низкоуровневым языком
* Компилируемым языком
